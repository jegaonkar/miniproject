C Programming Code To Create Pyramid and Pattern

Programs to print triangles using *, numbers and characters
Program to print half pyramid using * 
*
  * *
  * * *
  * * * *
  * * * * *
Source Code 
#include
  int main() {
      int i, j, rows;
      printf("Enter number of rows: ");
      scanf("%d", &rows);
      for (i=1; i<=rows; ++i) {
          for (j=1; j<=i; ++j)
          { printf("* "); }
          printf("\n");
      }
      return 0;
  }
